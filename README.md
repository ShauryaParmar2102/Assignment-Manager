# 📝 Assignment Manager

A simple **web-based assignment tracker** built with **HTML, CSS, and JavaScript**.  
Keep track of your tasks, mark them as completed, edit or delete them, and never miss a due date again!  

---

## **Features**
- Add new assignments 📝  
- Mark assignments as **Pending** or **Completed** ✅  
- **Edit** assignment names ✏️  
- **Delete** assignments 🗑️  
- Data is stored in **localStorage** so it persists across page refreshes

## **💻 How to Download & Run **

You can get the project files directly from GitHub:

1. Go to this repository on GitHub.

2. Click the green Code button at the top right.

3. Choose Download ZIP.

4. Extract the ZIP (if you downloaded it) and open the folder.

5. Open index.html in your browser to run the Assignment Manager.
